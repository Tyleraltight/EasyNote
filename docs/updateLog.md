# EasyNote Update Log

---

## V1.2.5 — 2026-03-16

### New Features
- **新增 Google 多元登录**：支持通过 Google 账号一键无感登录，且数据与原 GitHub 账号基于多账号体系平级管理
- **PWA 静默更新机制**：App 在切回前台时自动检查更新并在有新版本时静默刷新应用体验，告别手动重载

## V1.2.4 — 2026-03-13

### New Features
- **Flag 完成动画**：当年度 Flag 达成目标时，屏幕正中展示低调的 checkmark draw + fade 动画，2.5 秒后自动消失，不阻塞操作
- **待办置顶功能**：待办列表每条事项新增「图钉」按钮（垃圾桶左侧），已置顶项排列在列表顶部并以淡 amber 背景区分
- **更新日志卡片**：用户首次打开新版本时，屏幕居中展示更新内容卡片，点击空白处关闭，基于版本号通过 localStorage 控制仅显示一次

### Bug Fixes
- **日历打卡记录丢失**：修复重置已完成 Flag 时 `history` 被清空的问题，重置现在只归零 `current`，保留所有历史打卡记录

### Dependencies
- 新增 `framer-motion` 用于更新日志卡片动画
